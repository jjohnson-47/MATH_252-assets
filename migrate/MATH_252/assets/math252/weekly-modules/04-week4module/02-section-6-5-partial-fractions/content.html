  • Integrate the resulting simplified fractions using basic antiderivative rules.

The Big Picture: Breaking Down Complex Fractions

Partial fraction decomposition is a reverse engineering technique. Instead of adding fractions to get a complex rational function, we break down a complex rational function into simpler fractions that we can integrate easily.

Analogy: Just as we can factor polynomials to reveal their building blocks, partial fraction decomposition reveals the "building block fractions" that, when added together, give us our original rational function.

$$\frac{3x + 1}{x^2 - 1} = \frac{A}{x-1} + \frac{B}{x+1}$$

Once we find $A$ and $B$, each simple fraction integrates easily!

When to Use Partial Fraction Decomposition

Conditions for Partial Fraction Decomposition

Use partial fractions when:

  1. The integrand is a rational function: $\frac{P(x)}{Q(x)}$ where $P(x)$ and $Q(x)$ are polynomials
  2. The degree of the numerator is less than the degree of the denominator (proper fraction)
  3. The denominator can be factored into linear and/or irreducible quadratic factors

If degree of numerator ≥ degree of denominator: Use polynomial long division first to get:

$$\frac{P(x)}{Q(x)} = \text{Polynomial} + \frac{\text{Remainder}}{Q(x)}$$

Then apply partial fractions to the proper fraction remainder.

The Three Cases of Partial Fraction Decomposition

Case Factor Type Decomposition Form Example
Case 1 Distinct Linear Factors $\frac{A}{x-a} + \frac{B}{x-b}$ $\frac{1}{(x-1)(x+2)} = \frac{A}{x-1} + \frac{B}{x+2}$
Case 2 Repeated Linear Factors $\frac{A}{x-a} + \frac{B}{(x-a)^2} + \frac{C}{(x-a)^3}$ $\frac{1}{(x-1)^3} = \frac{A}{x-1} + \frac{B}{(x-1)^2} + \frac{C}{(x-1)^3}$
Case 3 Irreducible Quadratic Factors $\frac{Ax + B}{x^2 + px + q}$ $\frac{1}{(x-1)(x^2+1)} = \frac{A}{x-1} + \frac{Bx + C}{x^2+1}$

Fundamental Examples

Example 1: Case 1 - Distinct Linear Factors

Problem: Evaluate $\int \frac{3x + 1}{x^2 - 1} \, dx$.

Step 1: Factor the denominator and set up partial fractions.

$x^2 - 1 = (x-1)(x+1)$

$$\frac{3x + 1}{(x-1)(x+1)} = \frac{\textcolor{blue}{A}}{x-1} + \frac{\textcolor{red}{B}}{x+1}$$

Step 2: Clear denominators by multiplying both sides by $(x-1)(x+1)$.

$$3x + 1 = \textcolor{blue}{A}(x+1) + \textcolor{red}{B}(x-1)$$

Step 3: Find the unknown coefficients using strategic substitution.

Method 1: Strategic values

Let $x = 1$: $3(1) + 1 = \textcolor{blue}{A}(1+1) + \textcolor{red}{B}(1-1)$ → $4 = 2\textcolor{blue}{A}$ → $\textcolor{blue}{A = 2}$

Let $x = -1$: $3(-1) + 1 = \textcolor{blue}{A}(-1+1) + \textcolor{red}{B}(-1-1)$ → $-2 = -2\textcolor{red}{B}$ → $\textcolor{red}{B = 1}$

Step 4: Write the partial fraction decomposition.

$$\frac{3x + 1}{x^2 - 1} = \frac{\textcolor{blue}{2}}{x-1} + \frac{\textcolor{red}{1}}{x+1}$$

Step 5: Integrate each term.

$$\int \frac{3x + 1}{x^2 - 1} \, dx = \int \frac{\textcolor{blue}{2}}{x-1} \, dx + \int \frac{\textcolor{red}{1}}{x+1} \, dx$$
$$= \textcolor{blue}{2}\ln|x-1| + \textcolor{red}{1}\ln|x+1| + C$$
$$= 2\ln|x-1| + \ln|x+1| + C$$

Example 2: Case 2 - Repeated Linear Factors

Problem: Evaluate $\int \frac{x^2 + 1}{(x-1)^3} \, dx$.

Step 1: Set up partial fractions for repeated linear factors.

$$\frac{x^2 + 1}{(x-1)^3} = \frac{\textcolor{blue}{A}}{x-1} + \frac{\textcolor{red}{B}}{(x-1)^2} + \frac{\textcolor{green}{C}}{(x-1)^3}$$

Step 2: Clear denominators.

$$x^2 + 1 = \textcolor{blue}{A}(x-1)^2 + \textcolor{red}{B}(x-1) + \textcolor{green}{C}$$

Step 3: Find coefficients using substitution and expanding.

Easy coefficient: Let $x = 1$: $1^2 + 1 = \textcolor{green}{C}$ → $\textcolor{green}{C = 2}$

Expand and compare coefficients:

$x^2 + 1 = \textcolor{blue}{A}(x^2 - 2x + 1) + \textcolor{red}{B}(x - 1) + \textcolor{green}{2}$

$x^2 + 1 = \textcolor{blue}{A}x^2 + (-2\textcolor{blue}{A} + \textcolor{red}{B})x + (\textcolor{blue}{A} - \textcolor{red}{B} + \textcolor{green}{2})$

Comparing coefficients: $x^2$: $1 = \textcolor{blue}{A}$ → $\textcolor{blue}{A = 1}$

$x^1$: $0 = -2\textcolor{blue}{A} + \textcolor{red}{B} = -2(1) + \textcolor{red}{B}$ → $\textcolor{red}{B = 2}$

Step 4: Write the decomposition and integrate.

$$\frac{x^2 + 1}{(x-1)^3} = \frac{\textcolor{blue}{1}}{x-1} + \frac{\textcolor{red}{2}}{(x-1)^2} + \frac{\textcolor{green}{2}}{(x-1)^3}$$
$$\int \frac{x^2 + 1}{(x-1)^3} \, dx = \int \frac{\textcolor{blue}{1}}{x-1} \, dx + \int \frac{\textcolor{red}{2}}{(x-1)^2} \, dx + \int \frac{\textcolor{green}{2}}{(x-1)^3} \, dx$$
$$= \textcolor{blue}{\ln|x-1|} + \textcolor{red}{2 \cdot \frac{(x-1)^{-1}}{-1}} + \textcolor{green}{2 \cdot \frac{(x-1)^{-2}}{-2}} + C$$
$$= \ln|x-1| - \frac{2}{x-1} - \frac{1}{(x-1)^2} + C$$

Example 3: Case 3 - Irreducible Quadratic Factor

Problem: Evaluate $\int \frac{2x^2 + 1}{(x-1)(x^2+1)} \, dx$.

Step 1: Set up partial fractions with linear and quadratic terms.

$$\frac{2x^2 + 1}{(x-1)(x^2+1)} = \frac{\textcolor{blue}{A}}{x-1} + \frac{\textcolor{red}{Bx + C}}{x^2+1}$$

Note: For irreducible quadratics, use $\textcolor{red}{Bx + C}$ in the numerator.

Step 2: Clear denominators.

$$2x^2 + 1 = \textcolor{blue}{A}(x^2+1) + (\textcolor{red}{Bx + C})(x-1)$$

Step 3: Find coefficients.

Strategic substitution: Let $x = 1$: $2(1)^2 + 1 = \textcolor{blue}{A}(1^2+1) + 0$ → $3 = 2\textcolor{blue}{A}$ → $\textcolor{blue}{A = \frac{3}{2}}$

Expand and compare:

$2x^2 + 1 = \frac{3}{2}(x^2+1) + (\textcolor{red}{Bx + C})(x-1)$

$2x^2 + 1 = \frac{3}{2}x^2 + \frac{3}{2} + \textcolor{red}{B}x^2 - \textcolor{red}{B}x + \textcolor{red}{C}x - \textcolor{red}{C}$

Comparing $x^2$ coefficients: $2 = \frac{3}{2} + \textcolor{red}{B}$ → $\textcolor{red}{B = \frac{1}{2}}$

Comparing constant terms: $1 = \frac{3}{2} - \textcolor{red}{C}$ → $\textcolor{red}{C = \frac{1}{2}}$

Step 4: Integrate the decomposed fractions.

$$\frac{2x^2 + 1}{(x-1)(x^2+1)} = \frac{\textcolor{blue}{\frac{3}{2}}}{x-1} + \frac{\textcolor{red}{\frac{1}{2}x + \frac{1}{2}}}{x^2+1}$$
$$\int \frac{2x^2 + 1}{(x-1)(x^2+1)} \, dx = \frac{3}{2}\int \frac{1}{x-1} \, dx + \frac{1}{2}\int \frac{x}{x^2+1} \, dx + \frac{1}{2}\int \frac{1}{x^2+1} \, dx$$
$$= \frac{3}{2}\ln|x-1| + \frac{1}{4}\ln(x^2+1) + \frac{1}{2}\arctan(x) + C$$

Mastering Partial Fraction Decomposition:

  • Factor first: Always completely factor the denominator before setting up partial fractions
  • Know the three cases: Linear, repeated linear, and irreducible quadratic factors each have specific forms
  • Strategic substitution: Use values that make terms zero to find coefficients efficiently
  • Polynomial division: If degree of numerator ≥ degree of denominator, divide first
  • Check your work: Add your partial fractions back together to verify they equal the original
  • Integration payoff: Each partial fraction becomes a basic integral (logarithmic, arctangent, or power rule)
