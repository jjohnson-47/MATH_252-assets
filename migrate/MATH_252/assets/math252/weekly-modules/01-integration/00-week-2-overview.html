Core Theorems & Examples

Fundamental Theorem of Calculus, Part 1

If \(F\) is an antiderivative of \(f\) on \([a,b]\), then \[ \int_{a}^{b} f(x)\,dx = F(b) - F(a). \] Part 1 justifies evaluating a definite integral using any antiderivative.

Fundamental Theorem of Calculus, Part 2

If \(f\) is continuous on \([a,b]\), then the function \(g(x)=\displaystyle\int_{a}^{x} f(t)\,dt\) is differentiable and \(g'(x)=f(x).\)

Mean Value Theorem for Integrals

There exists a \(c\) in \([a,b]\) such that \[ \int_{a}^{b} f(x)\,dx = f(c)(b-a). \] We use this result to understand average value and to estimate errors in numerical rules.
