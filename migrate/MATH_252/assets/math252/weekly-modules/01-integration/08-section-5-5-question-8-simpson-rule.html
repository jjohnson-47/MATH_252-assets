Section 5.5 Question 8 – Simpson's Rule from Graph

Problem Statement

Use the graph of function $g(x)$ to apply Simpson's Rule with $n = 6$ subintervals to approximate $\int_{-3}^{3} g(x) \, dx$.

Function Graph and Data

Graph of $g(x)$ on the interval $[-3, 3]$ with gridlines at 0.5 unit intervals

Graph Description

The function $g(x)$ is defined on the interval $[-3, 3]$ with the following properties:

  • The graph shows a red curve representing $g(x)$
  • Gridlines are visible at every 0.5 unit step, both horizontally and vertically
  • Only the values 1 are labeled on both x and y axes
  • The x and y axes use the same scale
  • The plot extends from $x = -3.5$ to $x = 3$ and from $y = -2.5$ to $y = 2.5$

Solution Using Simpson's Rule

Step 1: Determine Parameters and Read Values

Given parameters:

  • Interval: $[a, b] = [-3, 3]$
  • Number of subintervals: $n = 6$ (even, as required)
  • Step size: $\Delta x = \frac{b-a}{n} = \frac{3-(-3)}{6} = \frac{6}{6} = 1$

The x-coordinates: $x_i = a + i\Delta x$ for $i = 0, 1, 2, 3, 4, 5, 6$

$x_0 = -3$
$x_1 = -2$
$x_2 = -1$
$x_3 = 0$
$x_4 = 1$
$x_5 = 2$
$x_6 = 3$

Step 2: Read Function Values from Graph

By carefully reading the graph at each x-coordinate:

$i$ $x_i$ $g(x_i)$ Coefficient Weighted Value
0-321$1 \times 2 = 2$
1-214$4 \times 1 = 4$
2-1-0.52$2 \times (-0.5) = -1$
30-2.54$4 \times (-2.5) = -10$
41-2.52$2 \times (-2.5) = -5$
52-0.54$4 \times (-0.5) = -2$
632.51$1 \times 2.5 = 2.5$

Coefficient pattern for Simpson's Rule: 1, 4, 2, 4, 2, 4, 1

Step 3: Apply Simpson's Rule Formula

Simpson's Rule formula:

$$S_6 = \frac{\Delta x}{3}\left[g(x_0) + 4g(x_1) + 2g(x_2) + 4g(x_3) + 2g(x_4) + 4g(x_5) + g(x_6)\right]$$

Substitute the values:

$$S_6 = \frac{1}{3}\left[g(-3) + 4g(-2) + 2g(-1) + 4g(0) + 2g(1) + 4g(2) + g(3)\right]$$

Insert the function values:

$$S_6 = \frac{1}{3}\left[2 + 4(1) + 2(-0.5) + 4(-2.5) + 2(-2.5) + 4(-0.5) + 2.5\right]$$

Calculate step by step:

$$S_6 = \frac{1}{3}\left[2 + 4 - 1 - 10 - 5 - 2 + 2.5\right]$$
$$= \frac{1}{3}\left[6 - 1 - 10 - 5 - 2 + 2.5\right]$$
$$= \frac{1}{3}\left[5 - 17 + 2.5\right]$$
$$= \frac{1}{3}\left[-9.5\right]$$
$$= -\frac{9.5}{3} = -\frac{19}{6}$$

Final Answer:

$$\boxed{S_6 = -\frac{19}{6} \approx -3.167}$$

Interpretation of the Result

The negative result makes sense when we examine the graph:

  • The function $g(x)$ spends most of its time below the x-axis on the interval $[-3, 3]$
  • The area below the x-axis contributes negatively to the integral
  • While there are some positive areas (when $g(x) > 0$), the negative areas dominate
  • The result $-\frac{19}{6} \approx -3.167$ represents the net signed area

Important: This is a signed area, not the total area. If we wanted the total area bounded by the curve and x-axis, we would need to break the integral into regions where $g(x) > 0$ and $g(x) < 0$ and take absolute values.

Key Learning Points:

  • Graph reading skills: Carefully interpreting function values from a graph
  • Simpson's Rule application: Using the correct coefficient pattern (1-4-2-4-2-4-1)
  • Negative integrals: Understanding that integrals can be negative when the function is mostly below the x-axis
  • Exact fractions: Leaving the answer as $-\frac{19}{6}$ maintains precision
  • Even intervals: Simpson's Rule requires an even number of subintervals
