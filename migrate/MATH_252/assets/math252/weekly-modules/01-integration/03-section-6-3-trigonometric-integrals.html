The Big Picture: Strategy for Trigonometric Integrals

Trigonometric integrals require a strategic approach based on the specific combination of functions and powers involved. Unlike substitution or integration by parts, there's no single universal methodâ€”instead, we need a toolbox of techniques.

The key insight: We use trigonometric identities to transform complex trigonometric expressions into forms we can integrate using basic rules or substitution techniques.

Key Idea 6.3.2: Powers of Sine and Cosine

Consider $\int \sin^m x \cos^n x \, dx$ where $m, n$ are nonnegative integers. ðŸ“– See complete strategy in textbook

Case 1: If $m$ is odd, factor out $\textcolor{blue}{\sin x}$ and use substitution $\textcolor{red}{u = \cos x}$

$$\sin^m x = (\sin^2 x)^{(m-1)/2} \sin x = \textcolor{red}{(1 - \cos^2 x)^{(m-1)/2}} \textcolor{blue}{\sin x}$$

Case 2: If $n$ is odd, factor out $\textcolor{blue}{\cos x}$ and use substitution $\textcolor{red}{u = \sin x}$

$$\cos^n x = (\cos^2 x)^{(n-1)/2} \cos x = \textcolor{red}{(1 - \sin^2 x)^{(n-1)/2}} \textcolor{blue}{\cos x}$$

Case 3: If both $m$ and $n$ are even, use power-reducing identities:

$$\sin^2 x = \frac{1 - \cos(2x)}{2}, \quad \cos^2 x = \frac{1 + \cos(2x)}{2}$$

Key Idea 6.3.8: Powers of Tangent and Secant

Consider $\int \tan^m x \sec^n x \, dx$ where $m, n$ are nonnegative integers. ðŸ“– See complete strategy in textbook

Case 1: If $n$ is even, factor out $\textcolor{blue}{\sec^2 x}$ and use substitution $\textcolor{red}{u = \tan x}$

$$\sec^n x = (\sec^2 x)^{n/2-1} \sec^2 x = \textcolor{red}{(1 + \tan^2 x)^{n/2-1}} \textcolor{blue}{\sec^2 x}$$

Case 2: If $m$ is odd, factor out $\textcolor{blue}{\sec x \tan x}$ and use substitution $\textcolor{red}{u = \sec x}$

$$\tan^m x \sec^n x = (\tan^2 x)^{(m-1)/2} \sec^{n-1} x \sec x \tan x = \textcolor{red}{(\sec^2 x - 1)^{(m-1)/2} \sec^{n-1} x} \textcolor{blue}{\sec x \tan x}$$

Product-to-Sum Formulas

For integrals like $\int \sin(mx)\cos(nx) \, dx$, use these product-to-sum identities:

$$\sin(mx)\sin(nx) = \frac{1}{2}[\cos((m-n)x) - \cos((m+n)x)]$$
$$\cos(mx)\cos(nx) = \frac{1}{2}[\cos((m-n)x) + \cos((m+n)x)]$$
$$\sin(mx)\cos(nx) = \frac{1}{2}[\sin((m-n)x) + \sin((m+n)x)]$$

Fundamental Examples

Example 1: Applying Key Idea 6.3.2 (Odd Power of Sine)

Problem: Evaluate $\int \sin^3 x \cos^2 x \, dx$ using the textbook's systematic approach.

This follows Key Idea 6.3.2, Case 1 from your textbook.

Step 1: Identify the case from Key Idea 6.3.2.

We have $\int \sin^{\textcolor{red}{3}} x \cos^{\textcolor{blue}{2}} x \, dx$ where $m = 3$ (odd) and $n = 2$ (even).

Strategy: Since $m$ is odd, use Case 1 - factor out $\textcolor{blue}{\sin x}$ and substitute $\textcolor{red}{u = \cos x}$.

Step 2: Apply the textbook formula for odd $m$.

Rewrite $\sin^3 x = \sin^2 x \cdot \sin x = \textcolor{red}{(1 - \cos^2 x)} \textcolor{blue}{\sin x}$

$$\int \sin^3 x \cos^2 x \, dx = \int \textcolor{red}{(1 - \cos^2 x)} \cos^2 x \textcolor{blue}{\sin x} \, dx$$
$$= \int \textcolor{red}{(\cos^2 x - \cos^4 x)} \textcolor{blue}{\sin x} \, dx$$

Step 3: Apply the substitution $\textcolor{red}{u = \cos x}$, $\textcolor{blue}{du = -\sin x \, dx}$.

$$= \int \textcolor{red}{(u^2 - u^4)} \textcolor{blue}{(-du)} = -\int (u^2 - u^4) \, du$$
$$= -\int u^2 \, du + \int u^4 \, du$$

Step 4: Integrate and substitute back.

$$= -\frac{u^3}{3} + \frac{u^5}{5} + C = -\frac{\cos^3 x}{3} + \frac{\cos^5 x}{5} + C$$

Verification: You can check this result by differentiating!

Example 2: Both Powers Even

Problem: Evaluate $\int \sin^2 x \cos^2 x \, dx$.

Step 1: Identify the strategy.

Both powers are even, so we need power reduction formulas.

Step 2: Apply power reduction formulas.

$$\sin^2 x = \frac{1 - \cos(2x)}{2}, \quad \cos^2 x = \frac{1 + \cos(2x)}{2}$$
$$\int \sin^2 x \cos^2 x \, dx = \int \frac{1 - \cos(2x)}{2} \cdot \frac{1 + \cos(2x)}{2} \, dx$$

Step 3: Expand the product.

$$= \frac{1}{4} \int (1 - \cos(2x))(1 + \cos(2x)) \, dx$$
$$= \frac{1}{4} \int (1 - \cos^2(2x)) \, dx = \frac{1}{4} \int \sin^2(2x) \, dx$$

Step 4: Apply power reduction again.

$$\sin^2(2x) = \frac{1 - \cos(4x)}{2}$$
$$= \frac{1}{4} \int \frac{1 - \cos(4x)}{2} \, dx = \frac{1}{8} \int (1 - \cos(4x)) \, dx$$
$$= \frac{1}{8}\left(x - \frac{\sin(4x)}{4}\right) + C = \frac{x}{8} - \frac{\sin(4x)}{32} + C$$

Example 3: Applying Key Idea 6.3.8 (Powers of Tangent and Secant)

Problem: Evaluate $\int \tan^2 x \sec^4 x \, dx$ using the textbook's approach.

This follows Key Idea 6.3.8, Case 1 since $n = 4$ is even.

Step 1: Identify the case from Key Idea 6.3.8.

We have $\int \tan^{\textcolor{blue}{2}} x \sec^{\textcolor{red}{4}} x \, dx$ where $m = 2$ (even) and $n = 4$ (even).

Strategy: Since $n$ is even, use Case 1 - factor out $\textcolor{blue}{\sec^2 x}$ and substitute $\textcolor{red}{u = \tan x}$.

Step 2: Apply the textbook formula for even $n$.

Rewrite $\sec^4 x = \sec^2 x \cdot \sec^2 x = \textcolor{red}{(1 + \tan^2 x)} \textcolor{blue}{\sec^2 x}$

$$\int \tan^2 x \sec^4 x \, dx = \int \tan^2 x \textcolor{red}{(1 + \tan^2 x)} \textcolor{blue}{\sec^2 x} \, dx$$
$$= \int \textcolor{red}{(\tan^2 x + \tan^4 x)} \textcolor{blue}{\sec^2 x} \, dx$$

Step 3: Apply substitution $\textcolor{red}{u = \tan x}$, $\textcolor{blue}{du = \sec^2 x \, dx}$.

$$= \int \textcolor{red}{(u^2 + u^4)} \textcolor{blue}{du} = \int u^2 \, du + \int u^4 \, du$$

Step 4: Integrate and substitute back.

$$= \frac{u^3}{3} + \frac{u^5}{5} + C = \frac{\tan^3 x}{3} + \frac{\tan^5 x}{5} + C$$

Example 4: Product-to-Sum Formulas

Problem: Evaluate $\int \sin(3x) \cos(5x) \, dx$ using product-to-sum identities.

This technique is essential for analyzing sound waves and signal processing applications.

Step 1: Apply the product-to-sum formula.

$$\sin(mx) \cos(nx) = \frac{1}{2}[\sin((m-n)x) + \sin((m+n)x)]$$

With $m = 3$ and $n = 5$:

$$\sin(3x) \cos(5x) = \frac{1}{2}[\sin((3-5)x) + \sin((3+5)x)] = \frac{1}{2}[\sin(-2x) + \sin(8x)]$$
$$= \frac{1}{2}[-\sin(2x) + \sin(8x)]$$

Step 2: Integrate term by term.

$$\int \sin(3x) \cos(5x) \, dx = \frac{1}{2} \int [-\sin(2x) + \sin(8x)] \, dx$$
$$= \frac{1}{2}\left[\frac{\cos(2x)}{2} - \frac{\cos(8x)}{8}\right] + C$$
$$= \frac{\cos(2x)}{4} - \frac{\cos(8x)}{16} + C$$
