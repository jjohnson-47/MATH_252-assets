This section introduces a fundamental method for approximating definite integrals using rectangles, leading to the concept of Riemann sums.

In this section, we'll explore how to approximate the area under a curve using rectangles, and how these approximations lead to the exact value of a definite integral through limits.

Definition: Riemann Sum

For a function \(f(x)\) defined on a closed interval \([a,b]\), a Riemann sum is formed by:

  • Partitioning the interval into \(n\) subintervals
  • Creating rectangles whose heights are determined by evaluating \(f(x)\) at some point in each subinterval
  • Summing the areas of these rectangles: \[\sum_{i=1}^n f(x_i)\Delta x\]

Key Methods for Rectangle Heights

There are three common ways to choose rectangle heights:

  • Left Hand Rule: Evaluate \(f(x)\) at the left endpoint of each subinterval
  • Right Hand Rule: Evaluate \(f(x)\) at the right endpoint of each subinterval
  • Midpoint Rule: Evaluate \(f(x)\) at the midpoint of each subinterval

Example: Computing a Riemann Sum

To approximate \(\int_0^4 x^2\,dx\) using 4 equal subintervals and the Right Hand Rule:

With \(\Delta x = 1\), we evaluate at \(x = 1, 2, 3, 4\): \[\sum_{i=1}^4 f(i)\Delta x = (1^2 + 2^2 + 3^2 + 4^2)(1) = (1 + 4 + 9 + 16)(1) = 30\]
