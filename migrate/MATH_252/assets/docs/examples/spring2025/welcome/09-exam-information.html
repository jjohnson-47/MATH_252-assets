Exam Information

Midterm Exam Format
  • Timed exam (not proctored)
  • Contains two parts:
    • Multiple choice questions
    • Written submission questions
  • Must be submitted through Blackboard
Final Exam Requirements
  • Proctoring Required
  • Remote students must use UAA Honorlock proctoring software
  • Must be submitted through Blackboard
Important Notice!

For remote proctoring setup, please review the Honorlock instructions well in advance of the exam: UAA Honorlock Setup Guide

Setup must be completed before the exam day to ensure a smooth testing experience.
