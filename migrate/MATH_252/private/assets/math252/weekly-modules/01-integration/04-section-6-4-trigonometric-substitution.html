Three Key Cases for Trigonometric Substitution

We encounter three main forms that suggest trigonometric substitution:

  • For \(\sqrt{a^2-x^2}\): Use \(x = a\sin\theta\)
  • For \(\sqrt{a^2+x^2}\): Use \(x = a\tan\theta\)
  • For \(\sqrt{x^2-a^2}\): Use \(x = a\sec\theta\)

Each substitution transforms our integrand into an expression involving only trigonometric functions, which we can then integrate using techniques from Section 6.3.

Essential Triangle Relationships

When using trigonometric substitution, it's helpful to draw a reference triangle. For example, when \(x = a\sin\theta\):

  • \(x = a\sin\theta\)
  • \(\sqrt{a^2-x^2} = a\cos\theta\)
  • \(\theta = \arcsin(\frac{x}{a})\)

These relationships come from the right triangle with hypotenuse \(a\) and opposite side \(x\).

Example 6.4.1: Integrating \(\int\sqrt{4-x^2}\,dx\)

Let's solve this step by step:

  1. Identify the form: This matches \(\sqrt{a^2-x^2}\) with \(a=2\)
  2. Make the substitution: \(x = 2\sin\theta\), so \(dx = 2\cos\theta\,d\theta\)
  3. Rewrite the radical: \[ \sqrt{4-x^2} = \sqrt{4-4\sin^2\theta} = 2\cos\theta \]
  4. Our integral becomes: \[ \int 2\cos\theta \cdot 2\cos\theta\,d\theta = 4\int \cos^2\theta\,d\theta \]
  5. Use the half-angle formula and integrate: \[ 4\int \cos^2\theta\,d\theta = 4(\frac{\theta}{2} + \frac{\sin(2\theta)}{4}) + C \]
  6. Substitute back: \(\theta = \arcsin(\frac{x}{2})\)

Mastering Trigonometric Substitution:

  • Pattern recognition: √a²−x² → sine, √a²+x² → tangent, √x²−a² → secant
  • Reference triangles: Essential for converting back to algebraic form
  • Domain considerations: Pay attention to angle restrictions and sign choices
  • Complete the process: Always convert back to the original variable
  • Practice makes perfect: These become intuitive with repeated application
