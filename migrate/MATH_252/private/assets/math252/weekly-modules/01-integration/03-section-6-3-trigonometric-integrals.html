Key Concept: Trigonometric Identities

To integrate trigonometric functions effectively, we'll rely heavily on these fundamental identities:

  • Power reduction: \(\sin^2 x = \frac{1 - \cos(2x)}{2}\)
  • Power reduction: \(\cos^2 x = \frac{1 + \cos(2x)}{2}\)
  • Product to sum: \(\sin A \cos B = \frac{1}{2}[\sin(A-B) + \sin(A+B)]\)

Integration Strategy

When integrating powers of sine and cosine, follow these steps:

  1. If you have an odd power of sine or cosine, factor out one term and use substitution.
  2. If you have even powers, use power reduction formulas to reduce the degree.

For example, to integrate \(\int \sin^3 x \, dx\):

\[ \int \sin^3 x \, dx = \int \sin x \cdot \sin^2 x \, dx = \int \sin x \cdot \frac{1-\cos(2x)}{2} \, dx \]

Example 6.3.1: Integrating \(\int \sin^2 x \, dx\)

Let's solve this step by step:

  1. Apply the power reduction formula: \[ \int \sin^2 x \, dx = \int \frac{1-\cos(2x)}{2} \, dx \]
  2. Separate the integral: \[ = \frac{1}{2}\int 1 \, dx - \frac{1}{2}\int \cos(2x) \, dx \]
  3. Evaluate each part: \[ = \frac{1}{2}x - \frac{1}{4}\sin(2x) + C \]

  • Substitution with odd powers: Factor out one function and substitute
  • Product-to-sum identities: Convert products of different trig functions
  • Reduction formulas: Use when repeated integration by parts is needed
  • Pattern recognition: Practice identifying which technique applies to each type
