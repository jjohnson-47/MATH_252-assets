Technology Troubleshooting Guide

Preventing Technical Issues

Pro Tip: Taking preventive measures can help you avoid most common technical problems.
  • Use Both Wired & Wireless: A wired Ethernet connection provides more stability, especially during online tests. Keep it as a backup option.
  • Keep Software Updated: Regularly check and install updates for your operating system, browsers, and applications.
  • Scan for Viruses: Maintain active antivirus protection and conduct regular system scans to prevent malicious software interference.

Quick Solutions

When Problems Occur: Try these simple fixes before seeking additional help.
  • Close & Reopen: If a program or webpage isn't working properly, close it completely and try again.
  • Switch Browsers: If Chrome isn't working, try Firefox (or vice versa). Keep multiple browsers installed.
  • Restart Device: A simple reboot can resolve many unexpected technical glitches.
  • Try Another Computer: If possible, test whether the issue persists on a different device.

Getting Help

Need Additional Support? Don't struggle alone - reach out for help.
  • Course Resources: Check the "Student Help" section in Blackboard for guides and FAQs.
  • KPC IT HelpDesk: Contact for technical assistance with any campus technology issues.
  • Document the Problem: When reporting issues, note:
    • What you were trying to do
    • What error messages appeared
    • What steps you've already tried
